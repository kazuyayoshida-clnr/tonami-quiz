# 砺波の土偶 謎解きクイズ

砺波正倉のPDFを読み込んで謎解きクイズを出題するWebアプリです。

## セットアップ

### 1. 依存パッケージをインストール
```bash
npm install
```

### 2. APIキーを設定
`.env.local` ファイルを開いて、Anthropic APIキーを入力してください：
```
ANTHROPIC_API_KEY=sk-ant-あなたのAPIキー
```

### 3. ローカルで起動確認
```bash
npm run dev
```
ブラウザで http://localhost:3000 を開いて動作確認

### 4. Vercelにデプロイ
```bash
vercel
```

## 使い方
1. 砺波正倉院フォルダのPDFをドラッグ＆ドロップ
2. 「謎解きを始める」をクリック
3. 土偶キャラがチャット形式で10問出題
4. 4択で回答して正解・解説を確認
